# unannounced-filter

Generate a list of prefixes that are not announced by any ASN, based on the latest BGP dump from [https://thyme.apnic.net](https://thyme.apnic.net). The output is a list of CIDR blocks that can be used for filtering unannounced IP addresses.

[Current blacklist output generated by GitHub Actions here](https://raw.githubusercontent.com/BadAimWeeb/unannounced-filter/refs/heads/master/blacklist.txt)

Note: Private ranges, CGNAT range and DS-Lite range are not included in the output, as they are dependent on the internal network configuration and may be used for legitimate purposes.

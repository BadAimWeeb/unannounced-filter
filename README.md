# unannounced-filter

Generate a list of prefixes that are not announced by any ASN, based on the latest BGP dump from [https://thyme.apnic.net](https://thyme.apnic.net). The output is a list of CIDR blocks that can be used for filtering unannounced IP addresses.

[Current blacklist output generated by GitHub Actions here](https://raw.githubusercontent.com/BadAimWeeb/unannounced-filter/main/blacklist.txt)
